## Primary States

1. New Session
* Previous Session
* Saved Session

## Navigation Paths

* Visits page for first time
  * Uses peteshow to fill out a form once
  * Saves current data
* Visits page a second time
  * Fills out form using:
    * new random data
    * data from previous session
  * Saves current data
* Visits page a third time
  * Fills out form using:
    * new random data
    * data from previous session
    * data from a saved session
